##  #️⃣ О проекте
Marea — это веб-маркетплейс, в котором реализованы ключевые функции для взаимодействия пользователей с платформой: регистрация, работа с товарами, корзина и поиск.

---

## ✨ Возможности

🔐 Регистрация пользователей <br>
Создание полноценного аккаунта с сохранением в базе данных

🛒 Корзина товаров <br>
Добавление, удаление и управление товарами

🔍 Поиск товаров <br>
Быстрый поиск по доступным позициям

---

## ⚙️ Технологии
Frontend: HTML / CSS / JavaScript<br>
Backend: Python / FastAPI / SQLAlchemy / Pydantic / Passlib<br>
Database: PostgreSQL

---

## 🔧 Запуск проекта
```bash
git clone https://github.com/FunnyBird12/marea.git
cd marea

python -m venv venv
source venv/bin/activate  # venv\Scripts\activate

pip install -r requirements.txt

uvicorn get:app --reload
```
